# Workdesk

Common files, assets, and walkthroughs for setting up my work environment in OSX.

## Linking

```sh
ln -Fs workdesk/zshrc .zshrc
ln -Fs dotfiles/gitconfig .gitconfig
ln -Fs dotfiles/gitignore_global .gitignore_global

source ~/.zshrc
```
